Pastel lilac (90s-new) launcher theme
-------------------------------------

Palette
- Base background: `#0f0d16` (deep ink), overlay haze: `#181523`.
- Panels: `#161320`; cards/inputs: `#1c1728`; frames: `#2a2633`.
- Primary lilac: `#d9c3ff`; deep lilac: `#b7a1f2`.
- Accent pops: mint `#8bf6e1`, rose `#f6b1d0`, warm highlight `#ffd6a5`.
- Text: primary `#f4f2ff`, secondary `#b9b5c9`, muted `#9b95ae`.
- Alerts: warn `#f6d38a`, error `#ef8fa3`.

Usage
- App background stays dark ink; headers/bars use a subtle lilac gradient.
- Panels/cards use panel/card colors with the frame color for borders/dividers.
- Primary actions sit on the lilac accent with dark-ink text; secondary actions use panel/card tones.
- Inputs use the card color with frame borders; focus/selection highlights use the mint accent.
- Status/warning text uses warn/error tones; secondary labels use muted text.
